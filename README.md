## Setup

1. Download the latest stable neovim release: <https://github.com/neovim/neovim-releases/releases>
2. `chmod +x` the appimage file and move to `~/.local/bin` or elsewhere in your `PATH`
3. Clone this repo to `~/.local/nvim`
4. Profit
